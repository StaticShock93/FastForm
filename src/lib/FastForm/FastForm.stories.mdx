# FastForm

`FastForm` is a necessary component that eases the process of creating a form. One of the key
features of FastForm is that it will manage your forms state. You initialize and define your event handlers
within the props of FastForm componenet.It gives you a set of predefined and reusable event handlers and validators as well as giving you the option of defining
your own validators.
